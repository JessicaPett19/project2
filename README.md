# project2
3-Round Aim Trainer game made by Jessica Pettinicchi

This game was inspired by a project done by Marek Andrzejewski, by aimtrainer.io, and by a multitude of other aim trainers. The "duck" special targets were inspired by the old arcade carnival games of shooting duck targets going across water. 

Each level gets a little bit harder, as the targets move in new ways, and the special targets become dangerous.
